Choose the number of rulesets to render

You can choose one or two rulesets (default). Press "Remove second ruleset" checkbox to remove the second ruleset. Dropdowns and sliders remain but ruleset will be removed. I would advise doing this initially to experiment with just one to see what happens.

Choose a Ruleset

There are a large number of rulesets to choose from, including the Koch curve, Hilbert curve, the dragon curve, and--I think my personal favorite--one called ADH231a. The ruleset data is contained in the ruleset.json file.

Here is an example of what the data looks like (for the dragon curve).

"dragon": { "axiom": "FX", "rules": { "X": "X+YF+", "Y": "-FX-Y", "length_factor": 1 }, "angle": 90, "length_factor": 1, "max_Level": 12, "author": "Paul Bourke" },

The turtle system is like a game you may have played as a child, "Simon Says." The axiom (in this case "FX") is the starting point, and the rules tell the computer exactly what to. The symbol F tell the computer to step forward one unit and then do "X."

  • "F" -> render shape and translate one step in the x direction. "f" -> translate one step in the x direction "+" -> rotate clockwise by the angle "-" -> rotate counterclockwise by the angle "[" -> push "]" -> pop

    • The rule "X" says do "X", turn righ

    When there are two fractals, you have the option of keeping three variables (x and y translation and grid length) the same (default) for the two L-systems or they can be independent.

    I have found that you can create some nice patterns with different shapes added to the same L-system, and therefore it is best to keep those 3 variables synchronized.
